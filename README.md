## Installation

Récupérer le repo via la ligne de commande

`git clone https://github.com/Ulrichcodi/IMC.git`

Se positionner dans le répertoire

`cd calcul-IMC`

Récupérer les dépendances

`npm install`

Et enfin lancer l'application

`npm start`
